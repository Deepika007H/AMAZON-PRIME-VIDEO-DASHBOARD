📊 AMAZON PRIME VIDEO DASHBOARD
📌 Project Overview

This project presents an interactive Amazon Prime Video Dashboard built using Power BI to analyze content distribution, ratings, genres, and trends.

The goal of this project is to perform Exploratory Data Analysis (EDA) and create meaningful visualizations that provide business insights about the platform's content strategy.

🎯 Business Objective

To analyze Amazon Prime Video’s content library and answer key business questions such as:

How many movies vs TV shows are available?

Which genres are most popular?

Which countries produce the most content?

How has content production changed over the years?
